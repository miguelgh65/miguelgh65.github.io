Predictor de Precios Inmobiliarios

Actualmente, estamos trabajando en un modelo de inteligencia artificial diseñado para predecir precios inmobiliarios utilizando datos reales. Haciendo uso de la API de Idealista, hemos obtenido una extensa cantidad de información relacionada con los precios de las casas, anuncios de viviendas, ubicaciones y demás características relevantes.

Tras un análisis exhaustivo, observamos correlaciones entre variables como tamaño, ubicación y precio de las viviendas. Una vez identificadas estas correlaciones, hemos procedido a realizar regresiones para estimar el precio de las casas. Estas regresiones se llevan a cabo mediante métodos clásicos y técnicas de deep learning.

Además de predecir el precio, hemos desarrollado un clasificador basado en imágenes que determina el tipo de vivienda. Por ejemplo, a partir de las fotografías de un inmueble, nuestro modelo puede distinguir si se trata de un piso, un dúplex, un chalet, entre otros.

En resumen, nuestra herramienta no sólo estima el valor de una propiedad, sino que también categoriza su tipo con base en sus imágenes. Estos avances ofrecen una ventaja significativa para profesionales del sector inmobiliario, brindándoles herramientas precisas para mejorar su toma de decisiones y proporcionar una mejor experiencia a sus clientes.
