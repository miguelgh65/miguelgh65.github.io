Automated Weekly Service Report

For years, the process of reporting hours dedicated to each client was manually carried out using Excel sheets. This task, though routine, consumed a significant amount of time and had a margin of error due to the manual nature of the process.

To enhance and optimize this process, I decided to automate the generation of these reports. Using metrics provided by Salesforce, I developed a system that automatically assigns hours dedicated to each client. Although this might sound simple, the task wasn't straightforward. Salesforce provides data based on time per case, not per client, which added an extra layer of complexity to the challenge.

To overcome this difficulty, it was necessary to design and develop robust code that made intensive use of the Python Pandas library. This tool was pivotal for manipulating, transforming, and analyzing the data, allowing for the correct allocation of hours per client. This process has not only optimized an essential task but also reduced the margin of error, providing more accurate reports and freeing up valuable time that can be used for other critical tasks.
